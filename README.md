## Steps to Run the Application ##

### Run Comparify Server ###

1. `cd server`
2. `mvn clean install`
3. `java -jar target\comparify-server.jar`

### Run Comparify Client ###

1. `cd client`
2. `npm i`
3. `npm run serve`
